# 📘 Assignment: Games in Python

## 🎯 Objective

Build a classic word-guessing Hangman game using Python strings, loops, conditionals, and user input. You'll practice fundamental programming concepts while creating an interactive game.

## 📝 Tasks

### 🛠️ Build a Hangman Game

#### Description
Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. Your game should handle word selection, track player progress, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in the format: _ _ _
- Track remaining incorrect guesses and display them to the player
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win or lose messages at the end
